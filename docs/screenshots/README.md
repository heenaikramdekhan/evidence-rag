# Screenshots

> ⚠️ The `.svg` files here are **hand-drawn mockups**, clearly labelled
> "MOCKUP", used as placeholders in the main README. Replace them with **real
> screenshots** of the running app for a stronger portfolio impression.

## How to replace with real screenshots
1. Run the app (`uvicorn app.main:app` + `npm run dev`) and open http://localhost:5173.
2. Capture each screen and save it here, overwriting the placeholder — keep the
   **same base name** so the README links keep working:

   | File           | Screen to capture                                         |
   |----------------|-----------------------------------------------------------|
   | `chat.png`     | A question with a cited answer + the SOURCES list          |
   | `inspector.png`| The **Inspect** drawer showing ranked chunks + score bars  |
   | `upload.png`   | The **Upload** dialog (mid-upload or after indexing)       |
   | `history.png`  | The **History** panel with a few past queries              |

3. Save as **`.png`** and update the extensions in `README.md`
   (`chat.svg` → `chat.png`, etc.), then delete the `.svg` placeholders.

Tip: on Windows, `Win + Shift + S` snips a region; crop to the app's 860px column.
